## ПОВЕДЕНЧЕСКИЕ ПРИНЦИПЫ (важнее всего)
- **Не выдумывать.** Не подставлять правдоподобные числа вместо реальных. Что не читается на чертеже или не задано — `null` + «требует уточнения». Неверная цифра в смете дороже отсутствующей.
- **Допущения — вслух.** Каждое принятое допущение писать явно. Если трактовок несколько — показать варианты, не выбирать молча.
- **Неясно — остановиться и спросить.** Не угадывать параметры (толщина, сорт, цены, ставки), а задавать конкретный вопрос.
- **Модель — для суждений, код — для расчётов.** Извлечение данных из чертежа (vision) — задача модели. Суммы, ФОТ, цена — считает код/формулы, а не «прикидка» модели. Длинные числовые столбцы модель не суммирует.
- **Минимум, без лишнего.** Решать поставленную задачу, без спекулятивных функций и абстракций сверх запрошенного.

## ПРАВИЛА РАСЧЁТА СЕБЕСТОИМОСТИ
Алгоритм раскладки и масс — в `.claude/skills/materials-calculation/`.

### Материалы — лист
- Стандартный лист: **1250×2500 мм**.
- **Исключение для Ст3: по умолчанию 1500×3000 мм** (площадь 4.5 м²). Нержавейка (AISI 304/430) — стандартный 1250×2500 мм.
- Стоимость заготовки = **доля листа (1/N)** × стоимость полного листа,
  где N — сколько заготовок фактически размещается на одном листе (соответствующего формата).
- **КИМ к листу НЕ применяется.** Точность раскладки уже учтена в доле 1/N.
- **Остаток листа — на склад, не в отходы.** Если деталь занимает, например, 2/3
  листа и вторая не помещается, в стоимость детали идёт только её фактическая
  доля (2/3). Оставшаяся часть листа считается складским остатком и в
  себестоимость текущей детали НЕ включается.

### Материалы — труба / профиль
- Стоимость = длина (м) × цена материала × **КИМ 1.3**.

### Плотности (г/см³)
- Ст3 — 7.85 · AISI 304 — 7.90 · AISI 430 — 7.70

### Цены металла (₽/кг)

### Трудозатраты — модель ставки
Ставка минуты в каждой ячейке стоимости — это **ФОРМУЛА**, а не константа.
Итоговая ставка минуты = **ставка персонала (ФОТ) + ставка станка (машино-время)**.

- **Ставка минуты = `[ФОРМУЛА — задаст пользователь]`** — вписать в каждую ячейку стоимости минуты.
- **Ставка персонала (₽/мин)** — только зарплата исполнителя операции. Текущие значения ниже.
- **Ставка станка (₽/мин)** — амортизация/стоимость работы оборудования. Задаёт пользователь; привязка станка к операциям — тоже за пользователем. Пока пусто.
- ФОТ-время операции = Σ времени; если не указано: лазер — по длине реза, сварка — по длине шва; не вычисляется — поле пустое.

### Себестоимость и цена
- Прямая себестоимость = материалы + расходники + ФОТ.
- **Цена продажи = себестоимость × 2.0.**
- Накладные расходы (20%) — НЕ применять. Отложено до v2 программы.

## JSON-СХЕМА — файл `[ОБОЗНАЧЕНИЕ]_материал.json`
```json
{
  "schema_version": "1.0",
  "наименование": "",
  "обозначение": "",
  "количество_шт": 0,
  "материал": { "тип": "", "вид": "", "сорт": "", "толщина_мм": 0 },
  "длина_мм": 0,
  "ширина_мм": 0,
  "площадь_м2": 0,
  "масса_заготовки_кг": 0,
  "ким": 0,
  "метраж_м": 0
}
```
- Единицы: `*_мм` — миллиметры, `*_м2` — м², `*_кг` — килограммы, `метраж_м` — метры.
- Лист: заполнять длину/ширину/площадь/массу, `метраж_м`=0, `ким`=0.
- Труба: заполнять `метраж_м` и `ким`=1.3, `толщина_мм` — стенка.
- Неопределённые поля — `null`.

### Сборки (изделие из нескольких деталей)

`parts` — массив объектов по схеме выше.
