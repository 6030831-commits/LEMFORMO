# ЛЕМ-СТИЛЬ — структура проекта Claude Code

Распакуйте содержимое в **корень вашего проекта** в VS Code. Дерево:

```
your-project/
├── CLAUDE.md                          # правила расчёта (главный файл, < 200 строк)
├── .gitignore                         # блокирует секреты и *.local.*
├── .mcp.json                          # MCP-серверы (Google Sheets)
└── .claude/
    ├── settings.json                  # разрешения, реестр
    ├── agents/
    │   ├── spec-analyzer.md           # чертёж → JSON
    │   └── cost-estimator.md          # JSON → расчёт себестоимости
    └── skills/
        ├── materials-calculation/
        │   └── SKILL.md               # алгоритм масс/раскладки
        └── work-calendar/
            └── SKILL.md               # календарь смен
```

## Установка (по шагам)

1. Скопируйте все файлы и папки в корень репозитория.
   Скрытые файлы (`.gitignore`, `.mcp.json`, `.claude/`) в VS Code видны —
   если нет, нажмите «Show Hidden Files» в проводнике.

2. Проверьте, что Claude Code видит правила:
   откройте терминал в папке проекта, запустите `claude`, спросите
   «какие у тебя правила расчёта себестоимости» — он должен пересказать CLAUDE.md.

3. Настройте MCP Google Sheets (файл `.mcp.json`):
   - создайте service account в Google Cloud, скачайте JSON-ключ;
   - впишите абсолютный путь к ключу в `SERVICE_ACCOUNT_PATH`;
   - откройте доступ к таблице календаря на email service account'а
     (вида `...@...iam.gserviceaccount.com`);
   - требуется установленный `uv` (для `uvx`): https://docs.astral.sh/uv/

4. Секреты НЕ коммитьте. `service-account.json`, `.env` уже в `.gitignore`.

## Проверка агентов

- `spec-analyzer` — даёт ему чертёж (PDF/фото), просит спецификацию → должен вернуть JSON.
- `cost-estimator` — получает JSON → выдаёт таблицу (материалы | цена | ФОТ).

Если агент не виден — проверьте, что файл лежит именно в `.claude/agents/`
и YAML-шапка (`---name: ...---`) не повреждена.

## Что осталось дозадать (плейсхолдеры в CLAUDE.md)

- Формула итоговой ставки минуты (персонал + станок).
- Ставки станка по операциям.
- Отдельные ставки персонала на гибку/резку пилой/шлифовку/сборку (пока «прочие» 10.52).
- Цены металла за погонный метр для труб; точная доля 1/N для листа.
- Нормы расхода и цены расходников.
- Накладные расходы 20% — отложены до v2.
